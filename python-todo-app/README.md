# 📝 Simple Python To-Do List

A lightweight command-line application built with Python to manage your daily tasks.

## ✨ Features
- **Add Tasks:** Create new tasks easily.
- **Track Progress:** View which tasks are completed.
- **Data Persistence:** Tasks are saved in a `todo.txt` file so you don't lose them.
- **Delete Tasks:** Remove unwanted tasks from your list.

## 🚀 How to Run
1. Ensure you have Python installed.
2. Open your terminal.
3. Run the following command:
   ```bash
   python todo_app.py
